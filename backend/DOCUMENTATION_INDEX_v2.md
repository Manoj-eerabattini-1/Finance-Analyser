## 🔑 Key Concepts

### 🤖 LLM-Enhanced Goal Planning (PRIMARY)
The core differentiator: Goals are automatically enhanced with:
- AI interpretation of user intent
- Automatic categorization
- Financial suggestions
- Budget breakdown
- Confidence scoring
- Works with or without API keys

### 🎯 Smart Categories
AI recognizes and maps goals to categories:
- Vehicle, House, Education, Vacation, Emergency Fund, Investment, Wedding, Retirement, Business, Other

### 💡 Intelligent Suggestions
Each goal gets context-aware suggestions:
- Category-specific tips
- Budget guidelines
- Timeline advice
- Savings strategies

### 📊 Financial Tracking
Track progress toward goals with:
- Transaction linking
- Income/expense categorization
- Monthly reports
- Goal progress percentage
